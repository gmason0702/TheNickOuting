# Choose payment processor

## Question

Which payment processor should handle the golf-tournament credit-card charge (flat fee per player in v1, but keep it extensible to variable pricing later)? Compare candidates (e.g. Stripe, Square, PayPal) on: fees, ease of integrating a hosted/embeddable checkout, payout to a personal/business bank account, and how cleanly it fits a near-zero-fixed-cost, low-volume (a few hundred transactions/year) use case. Land on one processor and the checkout approach (e.g. hosted Payment Link/Checkout vs custom Elements form).
